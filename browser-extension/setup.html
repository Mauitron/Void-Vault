Void Vault

Just a few steps to set up your secure password system

How Void Vault Works

Void Vault creates secure passwords for every website you visit:

  • Never stores passwords - generates them when you need them
  • Two-factor security: Your unique file + a phrase you remember
  • Different password for each website automatically
  • 100% private - everything stays on your device, works offline

Quick Setup

We'll run a simple setup wizard in your terminal. It takes about 1 minute.

Follow these steps:

  1. Open a terminal: Right-click your desktop ‚Üí "Open Terminal" (or press Ctrl+Alt+T)
  2. Run this command:
    ./target/release/void_vault
  3. Follow the wizard:
    • Choose your password style (shorter or longer input)
    • Choose character set (Standard recommended)
    • Type a training phrase (at least 20 characters)
    • Press ESC when done
  4. Come back here and click "I'm Done"

Create Your Configuration

Now you'll create your unique password configuration. This is a one-time setup.

Simple Steps:

  1. Open a terminal: Right-click your desktop ‚Üí "Open Terminal" (or press Ctrl+Alt+T)
  2. Copy and paste this command:
    ./target/release/void_vault
  3. Press Enter to run it
  4. Select option 2: Type 2 and press Enter
  5. Name: Type main and press Enter
  6. Description: Just press Enter (skip)
  7. Character set: Type 1 and press Enter
  8. Extra characters: Type 7 and press Enter
  9. Type a training phrase: Type at least 20 characters (can be anything)
  10. Press ESC when done typing
  11. Come back here and click "I'm Done"
üìù Note:

The phrase you type here is just for setup - it creates your unique configuration. You'll use different phrases for your actual passwords.

How to Use Void Vault

  1. Visit any website with a password field
  2. Click on the password field
  3. Press Ctrl+Shift+S to activate Void Vault
  4. Type your input phrase
